# Description

This is the API of Interact, a simple tool for easy interaction with smart contracts on Ethereum.

# Running locally

## Preparations

1. Install dependencies

```
npm install
```

2. Generate `.env` file

```
cp .env.example .env
```

3. Fill out variables in `.env`

## Launch
```
npm start
```

Application will be launched on http://localhost:3000.
